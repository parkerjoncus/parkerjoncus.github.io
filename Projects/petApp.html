Pets for All Application


In one of my Interprofessional Project courses (IPRO), students from different majors team up to work on a project. This project was focused on user-centered design. We were tasked with creating an application with the user in mind. We went through many different design steps, to try to get into the user's shoe to make sure that the app would be something that they want and use rather than just something we think is cool. Our idea was to create an application that allowed shelters to rent out pets to people for a certain amount of time to play with. After talking to shelters, we quickly realized that we should put a little more focus on the adoption of pets, but still allow rental, just with an end goal the the people the rent hopefully adopt or foster. We never created a working application, just a prototype. I hope to be able to create a working application eventually, even if it is still just a prototype. I just think it would be cool to improve my coding skills and create an application. We did create a video presentation which can be viewed below.
